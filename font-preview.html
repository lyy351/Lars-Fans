📰 AO3 工作皮肤 · 常用字体预览

⚠️ 重要说明： AO3 工作皮肤（Work Skin）不能加载外部字体文件（如 Google Fonts 或 .ttf 链接），只能使用读者设备上已安装的字体。以下列出的字体均为 Windows / macOS / Linux 等系统的常见字体，实际显示效果取决于读者的本地安装。建议在 CSS 中提供多个后备字体（fallback）。

🔤 衬线字体（Serif）—— 适合正文、复古风格

Times New Roman

这是一段示例文字。The quick brown fox jumps over the lazy dog. 中文宋体效果取决于后备字体。

font-family: 'Times New Roman', serif;
Georgia

Georgia 是专为屏幕设计的衬线体，阅读舒适。示例：ABC abc 123 中文显示依赖后备。

font-family: Georgia, serif;
Garamond

Garamond 是经典书籍字体，风格古典。示例：The quick brown fox...

font-family: Garamond, serif;
宋体 (SimSun) / 华文宋体 (Songti SC)

这是最常见的中文衬线体。Windows 用户看到 SimSun，macOS 用户看到 Songti SC。

font-family: "SimSun", "Songti SC", serif;
思源宋体 (Source Han Serif / Noto Serif SC)

思源宋体是开源字体，macOS 可能内置，Windows 需安装。示例：思源宋体显示效果。

font-family: "Source Han Serif", "Noto Serif SC", serif;

🔡 无衬线字体（Sans-serif）—— 现代、清晰

Arial

Arial 是通用无衬线体。示例：ABCDEFG 中文后备字体。

font-family: Arial, sans-serif;
Helvetica

Helvetica 是经典设计，macOS 默认。示例：Helvetica 显示效果。

font-family: Helvetica, sans-serif;
Verdana

Verdana 字符宽大，屏幕阅读极佳。示例：Verdana 字体示例。

font-family: Verdana, sans-serif;
微软雅黑 (Microsoft YaHei)

Windows 默认中文字体，无衬线。示例：微软雅黑显示效果。

font-family: "Microsoft YaHei", sans-serif;
苹方 (PingFang SC)

macOS / iOS 默认中文字体，无衬线。示例：苹方字体显示。

font-family: "PingFang SC", sans-serif;

📝 等宽字体（Monospace）—— 代码、打字机效果

Courier New

Courier New 是传统等宽字体。示例：int main() { return 0; } 中文后备。

font-family: 'Courier New', monospace;
Source Code Pro

Source Code Pro 是开源编程字体。示例：console.log("hello");

font-family: 'Source Code Pro', monospace;

✍️ 手写/装饰字体（Cursive/Fantasy）—— 慎用，依赖本地

Comic Sans MS

Comic Sans MS 是手写风格。示例：Handwriting style.

font-family: 'Comic Sans MS', cursive;
Brush Script MT

Brush Script 是连笔手写体。示例：Brush Script.

font-family: 'Brush Script MT', cursive;
💡 使用建议：
- 在 AO3 工作皮肤中，将上述 `font-family` 直接应用于 `.userstuff` 或特定元素。
- 为了更好的兼容性，建议提供多个后备字体（例如：`"SimSun", "Songti SC", "Times New Roman", serif`）。
- 避免使用中文引号（如 “宋体”），应使用英文引号包裹英文字体名（如 `"SimSun"`），防止 AO3 过滤器拦截。
- 本页预览效果完全依赖你本地安装的字体，与实际在 AO3 上读者所见一致。
